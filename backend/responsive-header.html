صفحه اصلی

این یک نمونه از هدر ریسپانسیو است که با HTML، CSS و JavaScript خالص ساخته شده است.

در دسکتاپ، منوی ناوبری به صورت افقی نمایش داده می‌شود و در موبایل و تبلت، منوی همبرگر نمایش داده می‌شود.

ویژگی‌های هدر

• طراحی ریسپانسیو با استفاده از Media Queries

• انیمیشن نرم برای منوی موبایل

• پشتیبانی از RTL برای متن‌های فارسی

• بسته شدن منو با کلیک خارج از آن

• رنگ آبی مدرن (#007BFF)

تست ریسپانسیو

برای تست عملکرد ریسپانسیو، اندازه پنجره مرورگر را تغییر دهید:

• عرض بیشتر از 768px: منوی دسکتاپ نمایش داده می‌شود

• عرض کمتر از 768px: دکمه همبرگر نمایش داده می‌شود
